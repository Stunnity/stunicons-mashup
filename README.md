# Stunicons  Mashup
Mashup with Stunicons

## Get started

Clone the repo

```bash
git clone https://github.com/DivinIrakiza/stunicons-mashup.git
```

Install the dependencies

```bash
cd stunicons-mashup
npm install
```


Serve the app

```bash
npm run dev
```
And browse

```bash
http://localhost:5000
```

Tada !!
